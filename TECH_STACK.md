# Technology Stack

This document describes the key technologies used in this repository.

## Frontend
- **Server-rendered HTML templates** generated using Flask's Jinja2 engine
- **Plain CSS** for styling the forms and pages

## Backend
- **Python** with **Flask** providing the web API and HTML routes
- **SQLAlchemy** for the SQLite database
- **OpenAI GPT** for slide outline and content generation

## Export Engine
- **python-pptx** to programmatically create PowerPoint files

These tools work together to enable automated slide creation from a simple web interface.
